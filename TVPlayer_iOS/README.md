# TVPlayer iOS

电视直播播放器 — iOS 原生版（Swift + AVPlayer）

## 功能

- 电视频道直播播放
- 同名频道折叠，多线路切换
- 自动换线路（7秒未接通自动切换）
- 源管理（自定义 m3u/m3u8 源）
- 频道搜索 / 收藏
- 手势控制：滑动切换频道、亮度/音量调节

## 构建

使用 Xcode 打开 `TVPlayer_iOS/` 目录：

```bash
cd TVPlayer_iOS
xed .
```
